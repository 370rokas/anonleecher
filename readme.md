# AnonScraper

Scrapes uploaded files to AnonFile by provided query using Google Dorking <br><br>
![Main Menu][img_main]

## 🔧 Installation
Use the package manager [pip][url_pip] to install the requirements.

```bash
pip install -r requirements.txt
```

## ✨ Usage

```bash
anonscraper.py -q <query>
```
